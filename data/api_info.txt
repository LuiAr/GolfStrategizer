Field    Description
apiRequestsLeft    Number of API requests left
courseID    Course ID
numCoordinates    Number of coordinates
coordinates    Array of coordinates
- poi: point of interest (
    1= green, 
    2 = Green Bunker, 
    3 = Fairway Bunker, 
    4 = Water, 
    5 = Trees, 
    6 = 100 Marker, 
    7 = 150 Marker, 
    8 = 200 Marker, 
    9 = Dogleg, 
    10 = Road, 
    11 = Ladies' tee, 
    12 = Men's tee
)
- location: location of PoI (1 front, 2 middle, 3 back)
- sideFW: Side of fairway of PoI (1 left, 2 center, 3 right)
- hole : hole number of PoI
- latitude: Latitude of PoI
- longitude: Longitude of PoI
